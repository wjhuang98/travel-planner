# Travel Planner
This app allows users to search up places they would like to travel to and displays the top TripAdvisor results and their given information (names, addresses, etc.)

- OpenCage Geocoding

- React.js
- Vite
- TypeScript
- Tailwind CSS
- Go
- Redis
